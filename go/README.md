# GRID Admin

## Tech Stack

- **Web Framework**: [Gin](https://github.com/gin-gonic/gin)
- **ORM**: [GORM](https://gorm.io/)
- **Database**: PostgreSQL
- **Authentication**: JWT (JSON Web Token)
- **Password Encryption**: bcrypt

## Getting Started

### Prerequisites

- Go 1.20+
- PostgreSQL 12+

### Run the Server

```bash
cd go
go run main.go
```

The server will start at `http://localhost:8080`
