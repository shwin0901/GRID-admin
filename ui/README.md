# GRID Admin UI

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- shadcn/ui
- TanStack React Query

## Prerequisites

- Node.js >= 18
- pnpm >= 8

### Installing pnpm

```bash
# Using npm
npm install -g pnpm

# Or using corepack (Node.js 16.13+)
corepack enable
corepack prepare pnpm@latest --activate
```

## Installation

```bash
pnpm install
```

## Development

```bash
# Start development server
pnpm dev

# Build for production
pnpm build

# Run linter
pnpm lint

# Preview production build
pnpm preview
```
